# OpenGL Engine.
This engine was made using ["The Cherno OpenGL Tutorial"](https://www.youtube.com/playlist?list=PLlrATfBNZ98foTJPJ_Ev03o2oq3-GGOS2) on youtube;

## How to build
```
cmake .
make
./OpenGLProject
```
